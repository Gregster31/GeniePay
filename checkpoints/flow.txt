GENIEPAY APP FLOW

main.tsx:
Imports global CSS styles and RainbowKit styles
Creates React root and renders App component in StrictMode
Entry point that bootstraps the entire application

App.tsx:
Sets up QueryClient with 1-minute stale time for caching blockchain data
Creates provider hierarchy: WagmiProvider → QueryClientProvider → RainbowKitProvider → AuthProvider → BalanceProvider
Enables Wagmi (React hooks for Ethereum blockchain interactions)
Enables RainbowKit for wallet connection UI
Provides authentication and balance context to the app
Renders AppRouter component which handles all application routing


BULK PAY FLOW EXPLAINED:
- Contract: Singleton, no employee data stored.
- Employer: Prepares list off-chain, approves tokens, calls bulkPayERC20(token, addresses[], amounts[]).
- Employees: Receive tokens automatically; no interaction needed.
- Notes: Gas grows with employee count; employer pays all fees.

HISTORY FLOW EXPLAINED:
- Using Block Explorer API to fetch transaction history of the wallet (5 calls/second) (Only works on EVM chains)
